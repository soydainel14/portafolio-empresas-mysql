Política de Privacidad

Última actualización: 1 de enero de 2026

1. Responsable del tratamiento

Código Empresarial gestiona los datos recopilados mediante el formulario “Crear Código Empresarial Portafolio”. Para contacto: hola@portafolio.do.

2. Datos que recopilamos

  • Datos del negocio: nombre comercial, razón social, RNC, actividad, local/número en la plaza, tamaño aproximado.
  • Datos de contacto: responsable, cargo, teléfono (WhatsApp) y correo.
  • Datos operativos: frecuencia y tipos de productos utilizados, prioridades operativas.
  • Datos técnicos: IP y user-agent para prevención de fraude/abuso.

3. Finalidades

  • Crear y administrar tu código empresarial.
  • Asignar un ejecutivo y coordinar entregas programadas.
  • Evaluar y habilitar facilidades de pago cuando aplique.
  • Comunicar seguimiento por WhatsApp/correo relacionado con la solicitud.
  • Prevención de fraude y abuso (incluye reCAPTCHA).

4. Base legal

Tratamos los datos con base en tu consentimiento y en la ejecución de gestiones precontractuales/comerciales derivadas de la solicitud.

5. Conservación

Conservamos la información el tiempo necesario para gestionar tu solicitud y, si aplica, durante la relación comercial y obligaciones legales.

6. Compartición

No vendemos tu información. Podemos compartir datos con proveedores tecnológicos estrictamente necesarios (hosting, correo, WhatsApp API si se habilita) bajo medidas de seguridad.

7. Derechos

Puedes solicitar acceso, corrección o eliminación escribiendo a hola@portafolio.do.

8. Seguridad

Aplicamos medidas técnicas y organizativas razonables para proteger la información.

9. Cambios

Podemos actualizar esta política; la fecha de “Última actualización” indicará la versión vigente.
